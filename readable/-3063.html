  11. 11. Migrating a WinForm or WPF app to Core • For designers, map existing application into a new Core application 1. dotnet new winforms 2. Delete all files except the .csproj file 3. Open the .csproj file 4. Stop the generation of AssemblyInfo using: <GenerateAssemblyInfo>false</GenerateAssemblyInfo> 5. Maps files from existing application into the .NET Core application: <ItemGroup> <Compile Include="..[Your Existing App]***.cs" /> </ItemGroup> 6. Add any references to other projects or libraries • Now you can use the designers in the old project while running on Core in the new project
