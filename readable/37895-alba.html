Alba

Use the Alba library in your testing projects to quickly write declarative specifications against the HTTP endpoint of an ASP.Net Core application by simulating an HTTP request in memory. Inspired by the "Scenario" testing mechanism in FubuMVC. Alba supports ASP.Net Core >2.1 and is tested against both netcoreapp2.1 and netcoreapp2.1.
