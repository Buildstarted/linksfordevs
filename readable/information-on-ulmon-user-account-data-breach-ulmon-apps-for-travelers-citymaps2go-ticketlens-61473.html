INFORMATION ON ULMON USER ACCOUNT DATA BREACH — Ulmon - Apps for Travelers: CityMaps2Go & TicketLens

Reading time: 3 minutes
Posted here: 08 May 2020

https://www.ulmon.com/blogging/2020/5/4/information-on-ulmon-user-account-data-breach

This page informs users of a data security breach Ulmon became aware of on April 27, 2020.

What Happened?

On April 27, 2020, Ulmon became aware of a data security breach based on an analysis of a data file provided to us by a web security expert. Based on this analysis we believe that an unauthorized party stole data associated with certain Ulmon user accounts in January 2020. 

What Information Was Involved?

The stolen user account information includes usernames, email addresses, hashed passwords (only strongly bcrypt encrypted/hashed passwords have been stolen) and, in some cases, Twitter names, Facebook account urls, telephone numbers, and user “about” texts. Not all of these data elements have been present for all accounts. The investigation indicates that the information that was stolen did not include other data like saved/bookmarked places, lists, notes, or further information stored in Ulmon user accounts.

What We Are Doing

We are taking action to protect users and to inform the authorities. Specifically:

  • We informed all affected users by email.

  • We will require all other users who have not changed their passwords since the time of the theft to do so.

  • We performed a security review of our systems, applied short term measures to secure the systems and started a process of further security improvements. 

  • We informed the Austrian Data Protection Authority about the breach.

  • We continue to enhance our systems that detect and prevent unauthorized access to user accounts.

What Ulmon Users Can Do

While Ulmon already has taken action to help secure your account, we encourage Ulmon users to consider the following account security recommendations:

  • Change passwords for any other accounts on which users used the same or similar information as for the Ulmon account.

  • Review the Ulmon account for suspicious activity.

  • Be cautious of any unsolicited communications that ask for personal information or refer you to a web page asking for personal information.

  • Avoid clicking on links or downloading attachments from suspicious emails.

We value the trust our users place in us, and the security of our users remains a top priority.
