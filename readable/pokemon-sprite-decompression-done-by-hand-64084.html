The Bail Project is a non-profit organization that aims to pay bail for those faced with unaffordable cash bail. While bail is intended to be used as an incentive to return to trial, in the recent years it has been used unfairly as a form of pretrial detention. After bail is returned, The Bail Project uses the same money to pay additional bail for others. For more information on The Bail Project, please visit https://bailproject.org/ .
