The essence of quantum mechanics

by Alex Kritchevsky
Reading time: 12-16 minutes
Posted here: 30 Jul 2020

http://alexkritchevsky.com/2020/07/24/qm.html

Here’s what I know about QM. I’m trying to learn QFT and it helps to have this stuff compressed into the simplest possible representation.

Conventions: c=1, gμν​=diag(−,+,+,+), because this keeps the operators in the form we expect. I like to write Sx​ for ∇S.

1. QM but starting from the solutions

QM makes a lot more sense to me if you (a) handle everything relativistically from the start and (b) just assume the form of the wave function solutions instead of deriving them. If I had my way I’d start a quantum mechanics course with special relativity, followed by introducing the scalar wave function, like this:

Consider a function on spacetime with the form ψ(t,x)=eiS(t,x)/ℏ which assigns a complex phase to every point. It is fully determined by the action S(x,t), and in particular given an initial state ψ0​, is determined by the action gradient dS=Sμ​dxμ. This lets us compare quantum states by integrating over some path Γ:

Later on when potentials are involved we will need to be specific about the path of integration, but for now we can think of S as a scalar function that determines ψ everywhere.

Relativistic invariance insists that ψ have the same value in any reference frame, and −iℏ∂ψ=−iℏ(∂S)ψ=(St​,Sx​)ψ must be a covariant 4-vector. Contraction with ψˉ​ extracts the vector components: ⟨ψ∣−iℏ∂∣ψ⟩=ψˉ​(St​,Sx​)ψ=(St​,Sx​). Finally, ∣(St​,Sx​)∣=Sx2​−St2​​ must be a Lorentz-invariant scalar.

We recognize that iℏ∂t​=E^ and −iℏ∂x​=P^, the energy and momentum operators. The quantum mechanical operators apparently extract properties of S, but because S is packed inside an exponential, they extract them as eigenvalues: iℏ∂t​ψ=St​ψ. Our quantum-mechanical inner product and our operators are just tools for extracting properties of S, since ψ is the only thing we can directly operate on. When an equation like the Schrödinger equation contains a P^=−i∂x​ operator, it is just a skew way of writing Sx​=p​.

The exact values of ψ up to a phase, and therefore S up to a constant, are not physically observable.

For a free massive spinless particle the action gradient is Sμ​=pμ​, the four-momentum, with ∣p∣=m, the rest mass. In the absence of a potential this gives the wave function:

ψ=ei/ℏ∫pμ​dxμ=ei/ℏpμ​Δxμ=ei/ℏ(p​⋅x−Et)

which is a Fourier mode with momentum p. Time evolution via exponentiation of the Hamiltonian amounts to translating in t:

ψ(t+Δt)=ei/ℏH^Δtψ(t)=eΔt∂t​ψ(t)=ei/ℏ(p​⋅x−E(t+Δt))

(This uses the idea that exponentiating a differential operator translates in that coordinate: ea∂x​f(x)=f(x+a).)

When an initial state is not a pure Fourier mode, we expand it as a sum of Fourier modes. For instance, if at t=0 we measure an electron at x=0, then the initial state is ψ(0,x)=δ(x)=∫eip​⋅xdp​.

When potentials are involved, dS is modified. The electromagnetic field, for instance, enters as dS=p−qA. Depending on the field configuration we may no longer be able to easily integrate ∫dS: if A includes a current, then it contains a ‘line’ of divergence, and the path integral’s result will depend on how many times Γ circles this pole in spacetime. This causes the path integral to give different values based on the choice of path. Summing over these paths corresponds in QFT to summing over the number of photons that are exchanged (I think. Will work it out in detail when I get to QFT).

2. Correspondences

Many concepts are not far away when you start with the solutions:

Energy-momentum: For a free particle St​=E and Sx​=p​, and m=E2−p2​ is the relativistic rest energy/momentum relation. The wave function looks like ψ=ei/ℏ∫p​⋅dx−Edtψ0​. A high energy/momentum corresponds to a rapidly changing action, and thus to a wave function that is quickly rotating as you translate in time or space.

Path integration: Relative changes in S can be found by integrating: S(f)−S(i)=∫Γ​dS along any curve Γ from i to f, and ψ(f)=ei/ℏ(S(f)−S(i))ψ(i). Thus ei/ℏ(S(f)−S(i)) is the ‘transition matrix’ between any two states, along a given path. The total transition amplitude is a sum over all possible paths between two states. This extends handily to QFT’s path integrals when creation/annihilation of particles is included.

The roles of ℏ and i: S↦eiS/ℏ is the conversion from ‘action’ space to ‘phase’ space. ℏ changes units from action (energy × time) to radians; if we set ℏ=1 we are declaring that we measure action in radians. The resulting space after mapping by eiS is physically meaningful, because in some cases we’ll end up summing these phase factors from multiple starting states and seeing interference patterns. I suspect that the output space is the U(1) that is identified with the electromagnetic gauge field but am not sure. If so, I think it would be good to write REM​ instead of i, in order to avoid accidentally conflating the i factors from rotations in different spaces. (Actually that’s my stance on i in general.)

Angular momentum: The orbital angular momentum operator, L^z​=−iℏ∂ϕ​, does the same thing as P^=−iℏ∂x​ but for a wave function in spherical coordinates. The azimuthal angle term looks like ψ∼ei/ℏ(lz​ϕ−Et), and L^z​ψ=lz​ψ. The azimuthal quantum number lz​ (often written m) measures how many times ψ oscillates in a rotation of the polar angle ϕ; it is quantized precisely because the ϕ coordinate has a built-in periodicity. A z-angular momentum value of lz​ labels the number of periods the wave makes as you rotate ϕ in the z-plane.

Spin-21​: If lz​=1/2, then ψ±​∼ei/ℏ(±21​ϕ−Et) acts like a spinor (by modeling the spin as orbital angular momentum, and omitting the r and θ components). This function appears trivially unphysical, since it has different values at ϕ=0 vs ϕ=2π. The resolution is the fact that it’s only meaningful to use the wave function to compare states that are connected by a path – and for a spinor it’s correct that ⟨ψ(ϕ=2π)∣ψ(ϕ=0)⟩=−1. (This is a useful mental model but isn’t the full story. My next post will be a truly exhaustive exploration of spinors.)

Spin-1: A vector-valued wave function ψ​=(ψx​,ψy​,ψz​), where the terms transform according to physical rotations, is a spin-1 object. To consider its angular momentum we diagonalize L^z​ by changing bases to

(ψ+1​,ψ0​,ψ−1​)=(ψx​+iψy​,ψz​,ψx​−iψy​)

These are known as spherical tensors. These terms locally look like ei/ℏ(lz​ϕ+f(θ)−Et) for lz​∈(+1,0,−1). (Apparently vector-valued wave functions have to have no radial variation, though, or this decomposition wouldn’t work.) The 0 component of Lz​ spin corresponds to a vector-valued wave function pointing only in the z direction. ±1 components correspond to having x or y components, with the sign determined by their relative phase.

Note what it means to have spin 1: it’s not that it fixes the value of the angular momentum; rather, it specifies the different ways that the angular momentum can transform under rotation. The three choices determine whether ψ​ is in the z direction (lz​=0) or whether it has a positive or negative ‘rotational’ components in the xy plane (lz​=±1). Particularly, having angular momentum lz​=+1 means that the y component is advanced in phase compared to the x component – this is why the ‘ladder’ operator L^+​=L^x​+iL^y​ serves to increase the angular momentum, because it includes a factor of eiϕ:

By the way, photons are spin-1 particles, but to be in the lz​=0 state would mean require that there be a rest frame where their vector points the same way as their velocity – and they have no rest frame, so the lz​ value is forbidden. The remaining ±1 states correspond to photon polarizations.

The Schrödinger Equation: We can write Sx2​−St2​=−m2 as St​=m2+Sx2​​=m1+m2Sx2​​​ and expand as a Taylor series (note that ∣Sx​/m∣=∣p/m∣≪1) to get:

St​=m(1+21​m2Sx2​​+O((m2Sx2​​)2)≈m+2mSx2​​

Using our operator forms we get the free-particle Schrödinger equation:

Interpreting, this says that the time-derivative of the action is a constant (the mass) plus a term proportional to the kinetic energy, plus higher-order terms that vanish at low momentums.

The m term is normally ignored in non-relativistic QM. It corresponds to a constant change in phase along any path (and adds a constant term to the Lagrangian), but it drops out of any calculation if you (a) only integrate over time and (b) never create/annihilate particles.

Schrödinger with potential: The V term in the non-relativistic Schrödinger ends up next to the kinetic energy term: E^ψ≈(m+P^2/2m+V)ψ. Working backwards through the derivation, we figure that the constraint on S must be: St​−V=m2+Sx2​​. But there is no particular reason this would have a clean relativistic form, since we treat our potential non-relativistically anyway.

Nevetheless we can add to our interpretation: the role of a classical scalar potential V is to modify the phase change as a wave function translates in time, such that the particle acts like it has energy (E−V) instead of E. The role of the vector potential is to modify the momentum, p​↦p​−qA.

The relativistic version uses the 4-potential A=(ϕ,A). The electromagnetic wave function is something like ψ=ei/ℏ∫(p​−qA)⋅dx−(E−qϕ)]dt=ei/ℏ∫(∂μ​S−qAμ​)dxμ. Interpreting: the potential serves to modify the contributions of energy and momentum by a path-dependent phase, by changing the action gradient as ∂S↦∂S−qA. A particle traversing a potential acts like it has a modified 4-momentum, of p→p−qA.

The Lagrangian: The integral ΔS=∫dS can be parameterized by time as

ΔS=∫(Sx​⋅dx/dt−St​)dt=∫Ldt

L=dS/dt is the source of the (single-particle) Lagrangian, and is where the elementary form L=T−V comes from. For a free particle, Ldt=−mdτ, and ΔS=−∫mdτ. In a classical scalar potential with St​=E=T+V:

In classical mechanics often E=T+V and p​⋅v=2T, giving 2T−(T+V)=T−V.

Regardless of how we parameterize S=∫dS, applying stationary-action will give the classical trajectory. Feynman’s classic explanation of this is that all but the ‘stationary’ path – the choice of Γ such that δS/δΓ∣Γ​=0 – will exhibit destructive interference in the macroscopic limit, resulting in the laws of classical physics. Quantitatively, this means that in the classical limit as ℏ→0, the path integral is dominated by the minimal path:

ℏ→0lim​∫dΓeiS[Γ]/ℏ​=ℏ→0lim​∫d(ΔΓ)eiS[Γmin​+ΔΓ]/ℏ∼ℏ→0lim​eiS[Γmin​]/ℏ​

I don’t exactly know how to make that rigorous but it makes heuristic sense: as ℏ→0 the function oscillates infinitely rapidly, cancelling itself out in the integral over ΔΓ, but at least the minimal path, where δS/δΓ=0, oscillates less than the rest do. We could imagine expanding S as a Taylor series S=S[Γmin​]+(δS/δΓ)δS+…, but I really don’t know if that’s allowed.

Noether’s Theorem: Suppose there is some dynamical variable q that S depends on. Then we can locally approximate ΔS=S(q,…)+Sq​Δq, adding a phase to the wave function ψ→eiSq​Δq/ℏψ. This leaves physics unchanged if and only if Sq​ is a constant, such that this is a uniform global phase transformation.

But if q is a physical symmetry of the system, then it must lead to the same physics; therefore Sq​ is a constant throughout the system’s evolution. Sq​ is called the ‘Noether charge’ corresponding to the q symmetry. E is the charge associated with t; p​ for x, L for θ, etc. (Presumably q=SA​ but I’m not sure what that means yet.)

Gauge transformations:

Since physics is determined by an action integral like ∫dS−qAμ​dxμ, any integrable (exact) form added to A and will only affect the action in a path-independent way:

∫if​dS−q(Aμ​dxμ+dΛ)=Sif​−qΛif​−q∫if​Aμ​dxμ

This performs a gauge transform that doesn’t affect the relative amplitudes of different paths between i→f – only the reuslting phase. As such there is no way to observe this effect in a closed system, so the addition of dΛ is a free variable in the theory. However, it turns out to be important when considering interacting systems, in ways that I haven’t learned yet but will be very important in QFT.

The Born Rule: The probability of observing a state α∣ψ⟩ is proportional to ∣α∣2. My pet theory is that this is due to destructive interference between histories of the observer. See my previous post.

Summary

  1. QM makes a lot more sense if you just take as granted the fact that the wave function has the form ψ=eiS/ℏ.
  2. Operators and inner products are ways of extract properties of S.
  3. The Schrödinger equation for a free particle is a low-energy approximation of the statement that ∣∂S∣=m.
  4. The only free physical quantity in a wave function is the 4-vector ∂S, which measures which part of the variation in S is in the spatial vs timelike direction.
  5. Potentials enter by modifying ∂S, eg ∂S↦∂S−qA. ∫if​dS=S(f)−S(i) may no longer hold depending on the properties of A.
  6. Intrinsic angular momentum is a property of what kind of object the wave function’s value is (scalar, vector, spinor, etc).

Normally you have to unlearn QM to learn relativistic QM, but the relativistic version makes much more sense in the first place so why not start there?
