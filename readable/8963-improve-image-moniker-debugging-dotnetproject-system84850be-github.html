Improve image moniker debugging 

Adds `DebuggerDisplay` attributes for `ImageMoniker` and
`ProjectImageMoniker` that produce strings such as `SdkWarning` for
instances of these types in the debugger.

Previously the debugger would just show the type name.

These changes have no effect unless a debugging.
