jsoncsharp 1.0.0.11

Reading time: 7-8 minutes
Posted here: 17 Mar 2020

https://www.nuget.org/packages/jsoncsharp/

A command line tool to genreate C# classes from json files. These can be example json documents or json schema. In addition, you can also generate a schema from a example json file.

  • .NET CLI 
dotnet tool install --global jsoncsharp --version 1.0.0.11

This package contains a .NET Core Global Tool you can call from the shell/command line.

This package has no dependencies.
