When watching a video and change volume, a 'chrome.exe' box is displayed. How to fix it? - Google Chrome Help

This problem started very recently for me and is not something I encountered before. When watching a video on chrome, say in YouTube, or Netflix, and I go to change the volume, see the usual volume bar, but accompanied with the volume bar is a translucent black box that says 'chrome.exe' with a play/pause and skip buttons. This is even the case if a video is paused in another tab or in the background. If i am not watching a video -- that is, the no tabs are open -- there is no box. See the pictures for more details on this phenomena. 
I am wondering how to fix this so the box does not pop up when I am watching videos on chrome. I have tried to restart my computer, reset all chrome settings, and reinstall chrome to no success. Any help with this matter would be greatly appreciated!
Other Chrome Questions and Issues, Windows, Stable (Default)
  1. Go to chrome://flags/#hardware-media-key-handling
  2. Set Hardware Media Key Handling to Disabled
  3. Restart the browser to apply the changes.
