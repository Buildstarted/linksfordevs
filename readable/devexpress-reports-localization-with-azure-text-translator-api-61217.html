DevExpress Reports: Localization with Azure Text Translator API

by DevExpress
Watching time: 6 minutes
Posted here: 30 Apr 2020

If you’ve watched our first Report Localization-video, you know that v20.1 – set to ship in May 2020 – allows you and your end-users to quickly translate individual reports for global distribution. In this video, we’ll describe a v20.1 feature we did not have time to document in our first video – Specifically, we’ll describe how you can use Azure Text Translation API and your own custom localization service within the DevExpress Web End-User Report Designer.
