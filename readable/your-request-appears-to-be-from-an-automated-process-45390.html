Description

The RAM-friendly REST client. - Submit GET, POST, PATCH, and other HTTP requests. - Send multiple request in batches using collections. - Add multiple workspaces to better organize your work. - Reuse variables across your workspace using Environment variables. - Update variables automatically with results of a previous response using variable chaining. - Automatically verify the contents of your response payload using API tests. Features
