DotVVM Minutes #6: Migrate ASP.NET Web Forms apps to .NET Core

by DotVVM
Watching time: 40 minutes

https://www.youtube.com/watch?v=ivPZyiz3_H0


This episode is dedicated to ASP.NET Web Forms developers and shows an upgrade path for Web Forms to .NET Core using DotVVM. You'll see how to install DotVVM in an existing Web Forms application, run both technologies side by side in the same project, and start rewriting individual pages from ASP.NET Web Forms to DotVVM. When all pages are rewritten, it is possible to just switch the project to .NET Core.
