bradwellsb/blazor-contacts: A project template for OData API, IdentityServer4 auth, and server-side Blazor

This app is a complete contact management solution created by Bradley Wells using Blazor. It includes an OData Web API backend, an IdentityServer4 authentication server, and a server-side Blazor front-end targeting .NET Core 3.1. Use the solution as a template for creating your own real-world web applications.

This project is accompanied by several blog posts describing the various components.
