Adopting Blazor - running C# on the client side and building rich web apps in one tech stack you’ll love - Telerik UI for Blazor Webinar

Reading time: 2 minutes

https://www.telerik.com/webinars/blazor-ui/adopting-blazor

Building scalable, rich, interactive web applications requires a well-thought architecture and knowledge across your tech stack. Until recently, your tech stack had to be a mix of several technologies. Blazor, the newest iteration of ASP.NET, is revolutionary in this regard. You can now run C# or Visual Basic directly in your browser as a complete replacement for JavaScript!

Watch the recording of this webinar to see the latest advancements in web technology, including:

  • What Blazor is as a technology
  • What problems it will solve
  • Why Blazor is a great technology to migrate to, whether you are coming from an old generation .NET tech or from JavaScript
  • Why using Telerik UI for Blazor will shorten development cycles and benefit your entire team

Blazor is a technology from Microsoft that allow developers to create rich web applications. It can run entirely on the server as well as directly in the front-end via a complete .NET virtual machine on top of WebAssembly. Telerik UI for Blazor is the most extensive suite of UI components and tools for Blazor. It helps developers focus on their application and not reinventing the wheel by providing off-the-shelf widgets for many common scenarios.
