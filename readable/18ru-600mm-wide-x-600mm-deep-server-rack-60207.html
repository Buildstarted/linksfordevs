4Cabling 18RU 600mm Wide x 600mm Deep Freestanding Server Rack

The 4Cabling freestanding server rack has been designed with configuration, effectiveness, and cost in mind. The 4Cabling freestanding server rack is an effective solution for data management installers or customers who require quick installation and reliable equipment.

Made from powder coated solid steel and welded joints for added strength. The server rack is supplied fully assembled and fit to the 19" standard rack mount equipment. With solid top and base, removable side panels, and the lockable door for added security. This data rack provides full flexibility for customers who know exactly what they require of their network cabinet/ server rack. 4Cabling server rack is built to the standard and compatible with both leading vendor equipment manufacturers as well as OEM brands.

4Cabling also has a full range of 19"server rack cabinet accessories such as Blanking panel, Cable Management Rail, Sliding Shelf, Fixed Shelf, Replacement Mesh Door, PDU, and many others standard 19" rack mount server rack accessories to compliment the 4Cabling range of server rack.

The freestanding server rack cabinet is suitable for: housing small server, DVR, security equipment, routers, patch panels, switches, telephone system, network & communication equipment and many more.
