CSLA .NET Version 5.1.0 Release

by Rockford Lhotka
Reading time: 2 minutes
Posted here: 20 Feb 2020

https://blog.lhotka.net/2020/02/17/CSLA-Version-5.1.0-Release

17 Feb 2020

I am pleased to announce the release of CSLA .NET version 5.1.0.

This is a big enhancement release, with 429 commits included since the previous release. These commits include many from contributors in the community, and they are listed in the release notes. It is wonderful to work with such a great community!

Major areas of focus in the 5.1.0 release include:

  1. Blazor and Razor Components support for server-side and WebAssembly
  2. ASP.NET Core 3.0 and 3.1 support has been enhanced for MVC and Razor Pages
  3. Lots of performance and bug fix changes to the data portal
  4. Enhancements to the analyzers
  5. The Templates package now installs as expected for VS 2017 and 2019

Obviously these are just the highlights from those 429 commits. You can see all the detail in the release notes.
