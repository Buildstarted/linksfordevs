Serenity MVC / TypeScript App Platform

"Serenity is a ASP.NET MVC / TypeScript application platform which has been built on open source technologies. It aims to make development easier while reducing maintenance costs by avoiding boiler-plate code, reducing the time spent on repetitive tasks and applying best software design practices. Serenity is best suited to business applications with many data entry forms or administrative interface of public facing web sites. This doesn't mean that it can't be used for other types of applications. Some of its features can be helpful for any kind of application."
