RemedyBG by remedybg

RemedyBG is a 64-bit Windows debugger written from scratch with the goal of replacing the behemoth Visual Studio debugger. 

All the code for RemedyBG is written from scratch which allows for a great deal of flexibility with respect to the functionality that can be implemented. There are a lot of interesting features coming down the pipe after realizing the wealth of information that compiler emits.

Please use the message boards at https://remedybg.handmade.network/ or contact me via email (see README) for any comments, requests, and so forth.

Note that the source code is not provided as part of the distribution.
