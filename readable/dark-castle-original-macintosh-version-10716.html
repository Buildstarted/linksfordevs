This is the original version of Dark Castle, released by Silicon Beach Software in 1986. Several terrible ports of this game were released in the years following, which sort of gave this game a bad name. Don't be fooled though, original versions of games are often better versions. In my opinion the original is still a great game that's aged well, even all these years later.
