Virtual: What's new in Visual Studio for Mac for .NET Developers

Reading time: 1 minute
Posted here: 31 Mar 2020

https://www.meetup.com/SeattleMobileDevelopers/events/269755021

The user group is going Virtual during the current stay-at-home order. We are really excited to have Jon Galloway (https://twitter.com/jongalloway), Sr. PM for VS & .NET at Microsoft, to come talk about all of the latest features in Visual Studio for Mac for .NET developers. Regardless if you are building mobile, web, or any other type of .NET application there is surely something for you! Even if you aren' ton a Mac, come hang out an ask questions!

We will be using Microsoft Teams "Live Events". You don't need to have an account or install anything, just a browser will do!

We will start at 5PM with a brief opening from James & at 5:15PM Jon will kick off the session and answer questions live.
