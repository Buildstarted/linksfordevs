From ‘dotnet run’ to “Hello World” - Matt Warren Have you ever stopped to think about all the things that happen when you execute a simple .NET program? This talk will delve into the internals of the recently open-sourced .NET Core runtime, looking at what happens, when it happens and why. Making use of freely available tools such as 'PerfView', we'll examine the Execution Engine, Type Loader, Just-in-Time (JIT) Compiler and the CLR Hosting API to see how all these components play a part in making 'Hello World' possible. Matt is a Microsoft MVP based in UK, and a C# dev who loves nothing more than finding and fixing performance issues. He’s worked with Azure, ASP.NET MVC and WinForms on projects for storing government weather data, medical monitoring devices and an inspection system that ensured kegs of beer didn’t leak! He’s previously been an Open Source Contributor to BenchmarkDotNet and RavenDB, nowadays he mostly blogs about the ‘Internals’ of the .NET Runtime at http://mattwarren.org.
