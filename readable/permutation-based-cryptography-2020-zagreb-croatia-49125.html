SILC workshop

The Security and Implementation of Lightweight Cryptography (SILC) workshop takes place the day after PBC 2020. As there is an intersection between permutation-based and lightweight cryptography, we allow double submissions between SILC and PBC.
If it is the case, please specify it when submitting.

Abstract submission

Submissions must include the name of the speaker, a title and an extended abstract (up to 3 pages) or the full paper (up to 15 pages). Contributors can send their proposal to submission (at) permutationbasedcrypto.org.

If you prefer, please find here the official call for papers in PDF.
