Event description

Description

Join us on April 2 in the hub space at Microsoft's NERD center to watch the live streaming launch of Visual Studio 2019. Learn how Visual Studio 2019 is more productive, modern, and innovative, and take the latest version for a spin.

We'll kick off with the keynote at 12pm ET. Be sure to arrive early to grab some lunch and swag! We'll then continue to stream the entire day of events, so feel free to drop in throughout the day. If you're lucky, you might meet some Visual Studio for Mac and Xamarin PMs and engineers!

The full agenda is here: https://visualstudio.microsoft.com/vs2019-launch/ (note that all times are in PT)

Food will be provided. ID is required in order to sign in with security.
