Cutter

Cutter goal is to be an advanced FREE and open-source reverse-engineering platform while keeping the user experience at mind. Cutter is created by reverse engineers for reverse engineers.

Cutter is using radare2 as its core engine. Thus, allows access to thousands of features via the GUI or by using the integrated terminal.

Cutter provides enormous amount of different widgets and features to make your Reverse Engineering experience as comfortable as possible.

By default, Cutter is coming with multiple modern themes -- Light, Native and Dark.

Completely FREE and licensed under GPLv3

Built using Qt C++ and design best practices

Fully featured graph view as well as mini-graph for fast navigation

View and modify any file with a rich and powerful Hex View

Quickly write python scripts to automate tasks

Use Native or Python plugins to extend Cutter's core functionality

Add, remove and modify bytes and instructions

Great for automation, crypto algorithms and malware analysis

Fully featured theme editor for easy and user-friednly customization of Cutter

Move different views and windows around to get your most efficient layout

Cutter interface was translated by our community to many different languages
