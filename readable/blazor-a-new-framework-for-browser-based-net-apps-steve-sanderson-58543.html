Blazor is a new web UI framework that has become one of the headline features of .NET Core 3 and beyond. It lets you build rich interactive user experiences in C# and Razor, offering the benefits of a single-page app (SPA) framework while keeping all your code and data in .NET. It gives you a modern component-based architecture for powerful code reuse, first-class development tools, and all the productivity of .NET.
