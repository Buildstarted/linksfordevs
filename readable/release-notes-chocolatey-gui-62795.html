Release Notes Chocolatey Gui

Reading time: 7-9 minutes
Posted here: 12 Jun 2020

This covers changes for the "chocolateygui" package, which is available as FOSS.

NOTE: For commercial editions, please also refer to Licensed Release Notes.

0.17.1 (June 11, 2020)

BUGS

  • Language localization files are missing in 0.17.0 - see #778
  • Chocolatey GUI runs as a 32 bit process when running on a 64 bit machine - see #779

  • Search in all available repositories - see #588
  • Hide Download Counters - see #569
