Reading time: 13-17 minutes
Posted here: 08 Feb 2020

https://dotnet.microsoft.com/platform/support/policy/aspnetcore-2.1

The package lists below represents the supported ASP packages when using ASP.NET Core 2.1 on .NET Framework. The list represents the released versions at the time of posting. However, in all cases, the latest released version must be used to be in a supported configuration. For example, dotnet-ef.2.1.0.nupkg is listed but in the future, 2.1.1 may be released. At that time, 2.1.1 will become the minimum supported version of the package.
