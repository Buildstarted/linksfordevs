MatBlazor

MatBlazor - Material Design components for Blazor and Razor Components

Install MatBlazor library via nuget.

Install-Package MatBlazor

or

dotnet add package MatBlazor

Add @using MatBlazor in main _Imports.razor

Usage

MatBlazor components are self-supporting.

MatBlazor components for server-side Blazor (ex Razor Components)

Used EmbeddedBlazorContent library:

1) Startup.cs

2) _Host.cshtml (head section)
