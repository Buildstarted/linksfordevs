Migrating a WPF App to .NET Core 3 (1/5)

by Microsoft Visual Studio
Watching time: 19 minutes

https://www.youtube.com/watch?v=OYAkmB8cQrs&list=PLReL099Y5nRdG-LQ6OZSPECF-eXjgFNrW&index=1


End-to-end walk-through of porting a .NET Framework WPF app to .NET Core 3. The code sample can be found at https://github.com/dotnet/windows-desktop/tree/master/Samples/BeanTrader. The initial project state is in the NetFx directory and the final, migrated .NET Core project, is in the NetCore directory.
