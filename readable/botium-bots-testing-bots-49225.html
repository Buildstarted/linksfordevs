TESTING

Keeping an eye on the quality of your bot.

Testing 100x faster

Quality assurance for chatbots starts on API level. Testing conversations through an API is 100x faster and generates constantly feedback for the development lifecycle.

Testing End-to-End

After passing the API level, Botium moves on to the user interface. No matter if a bot implements a web, mobile or an audio interface.

THERE'S MORE

Solutions for every use case.

  • Security Testing

    Botium supports out of the box security testing based on the OWASP Top Ten.

  • GDPR Testing

    Together with GEISTWERT we developed a standard for chatbot GDPR testing.

  • Humanification

    A very common problem of many bots. They cannot handle typical errors made by humans. Botium could train them.

  • Link Asserter

    Your bot is sending links? Botium will check them. Starting with the HTTP response, ending at the linked content.

  • SQL Asserter

    Could Botium compare answers of your bot with a database? Sure, Botium could do that for you.

  • Live Chat

    The coolest way to add new testcases to Botium? Do a live chat with your bot and capture the conversation.

  • Connectors

    Botium Box supports all common providers. Dialogflow, Watson, MS Bot Framework, Alexa, Facebook Messenger,...

  • Testlabs

    For a real end-to-end testing experience, Botium Box integrates Cloud- and Onpremise-Devicelabs.

  • Versioning

    Testcases should be versioned. Botium Box connects to the Git repo of your choice to version testcases.

  • Zed Attack Proxy

    Botium supports Continuous Security Testing by sending all conversations through a ZAProxy.

  • Performance Testing

    Can your chatbot deal with multiple users? Start running Stress and Load Tests and find it out.

  • Testcase Generator

    Autogenerate testcases out of the interaction model of your chatbot. Works for Watson, Dialogflow and Alexa.
