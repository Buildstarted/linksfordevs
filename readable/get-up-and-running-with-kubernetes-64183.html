Get up and running with Kubernetes

Reading time: 2 minutes
Posted here: 13 Jul 2020

https://azure.microsoft.com/en-us/resources/kubernetes-ebook-collection/

Build, deliver, and scale container-based applications faster with Kubernetes. Learn the basics of Kubernetes, then discover how to easily deploy and manage containers at scale with Kubernetes on Azure.
With the Kubernetes collection, you’ll get three e-books that will help you gain the knowledge and hands-on experience necessary to get started with Kubernetes—all in one place.
Access the collection to get:

  • Kubernetes Up and Running - Learn to develop and deploy applications, create and run a simple cluster, and integrate storage into containerized microservices. 
  • Designing Distributed Systems - Explore distributed system concepts and get reusable patterns and practices for building distributed systems. 
  • Hands-On Kubernetes on Azure - Learn how to deploy and manage container clusters at scale on Azure using Azure Kubernetes Service (AKS). 

You’ll also gain access to additional content, including a video series by Kubernetes co-founder Brendan Burns, a learning path with interactive courses, hands-on Kubernetes workshops, and best practice webinars to help accelerate your adoption of Kubernetes.
