Launching docs.github.com - The GitHub Blog

Reading time: 2 minutes
Posted here: 01 Jul 2020

https://github.blog/2020-07-01-launching-docs-github-com/

We know how important learning is to your success using GitHub. When you’re using our products, we hope you’ll feel confident and supported with the tools you need to keep growing in your journey. That’s why we’re launching a single home for all of GitHub’s product documentation, docs.github.com.

What we’re launching

Until now, GitHub has had two sites for product documentation to support both users and integrators, help.github.com and developer.github.com. As our product offerings have evolved, we decided to create a single place where everyone can find the information they need, whether you’re a new developer signing up for your GitHub account or a seasoned developer in the community. Starting today, you can find everything you need to learn about GitHub products on docs.github.com!

This new site combines the product content that was on help.github.com and developer.github.com into a unified experience. With docs.github.com, you’re able to search across all of GitHub’s product content in one place. In the future, we’ll be rolling out language support for all of our product documentation, including content for integrators. We currently support Japanese, Simplified Chinese, Brazilian Portuguese, and Spanish.

The launch of docs.github.com is just the first step in creating a place where you can discover new tools and approaches. We’ll continue to expand on this experience so that you have all the tools you need to be successful with GitHub. Look for more improvements in the future!
