Linux GUI applications in the Windows Subsystem for Linux (WSL) survey

Hello Linux GUI app fans! This survey is by the engineering teams working on WSL at Microsoft. We're working on adding the ability to run Linux GUI apps on Windows machines using the Windows Subsystem for Linux. We would greatly appreciate a few minutes of your time to fill out this survey, as we'd love to know what Linux GUI apps you'd be interested in using on a Windows machine. Please answer the questions below and indicate which Linux GUI apps you'd like to run in order of priority (the first GUI app that you write about should be the most important to you) and lastly optionally let us know what are your use cases for WSL. For privacy policy info on the data we collect please see the "Privacy and Cookies" link at the bottom of this survey. Thank you! -The WSL team The survey will take approximately 4 minutes to complete. If you'd like to learn more about this work you can read about on the Command line blog here: https://devblogs.microsoft.com/commandline/the-windows-subsystem-for-linux-build-2020-summary/#wsl-gui
