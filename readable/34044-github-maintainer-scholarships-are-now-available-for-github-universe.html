GitHub Maintainer Scholarships are now available for GitHub Universe

GitHub takes pride in fostering a collaborative community that can come together—as individuals and in teams—to create the future of software and make a difference in the world. We couldn’t accomplish this without open source maintainers, which is why we are happy to provide free and subsidized tickets to GitHub Universe 2019 for active maintainers. 

We’re opening applications for our GitHub Maintainer Scholarship Program to help you get to San Francisco November 13-14 for the big event. These scholarships, and our Inclusivity Scholarships, are steps toward making GitHub’s events more accessible and inclusive for the open source community that makes GitHub everything it is today.

Submit your application by 12 pm PT on Thursday, August 29. You must be 18 or older to be eligible. If selected, we’ll email you by Wednesday, September 4, and we’ll need confirmation that you’re attending Universe by Friday, September 13.
