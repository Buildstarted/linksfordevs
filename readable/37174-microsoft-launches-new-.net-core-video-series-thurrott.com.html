Microsoft Launches New .NET Core Video Series - Thurrott.com

Posted on September 23, 2019 by Paul Thurrott in Dev with 2 Comments

Microsoft today released a new series of developer videos explaining the entire .NET Core stack.

“Surprise! A full series of C#, .NET Core, and ASP.NET 101 tutorials for newbies!” Microsoft’s Scott Hanselman tweeted. “Over 80 free YouTube videos!”

There are 10 videos series, each of which is comprised of several videos. Series include C# 101, .NET Core 101, ASP.NET 101, Xamarin 101, Desktop and .NET Core 101, Introduction to Containers (Docker), Entity Framework Core 101, Intro to Visual Studio, ML.NET, and .NET for Apache Spark 101.

Pretty sweet!
