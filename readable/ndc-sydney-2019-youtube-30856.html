NDC Sydney 2019 will be taking place at Hilton Sydney from 14-18 October. NDC Sydney is a five day event with 2 days of pre-conference workshops and 3 days of conference. The 5 paralell conference tracks will cover these topics: .NET - Agile - C++ - Cloud - Database - Design - DevOps - Embedded - Front-End Framework - Fun - Functional Programming - Gadgets - Internet of Things - JavaScript - Microsoft - Mobile - People - Programming Languages - Security - Techniques - Testing - Tools - UX – Web and more.
