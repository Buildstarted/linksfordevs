DataTier.Net DataTier.Net creates stored procedure powered data-tiers for C# developers and is an alternative to Entity Framework. https://github.com/DataJuggler/DataTier.Net DB Compare Compare two SQL Server databases and view a report showing any schema differences. Export database schema to XML for remote comparisons. https://github.com/DataJuggler/DBCompare

Install

https://datajuggler.com/Downloads/DBCompare.msi

XML Mirror XML Mirror uses reflection to make it simple to create C# Xml Parsers and Writers. https://github.com/DataJuggler/XMLMirror Transparency Maker Manipulate images using Bitmap Query Language or BQL for short. If you know SQL, you know BQL. https://github.com/DataJuggler/TransparencyMaker

Install

https://datajuggler.com/Downloads/TransparencyMaker.msi
