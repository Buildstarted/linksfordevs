Cheap USB Host Microcontroller [CH559, ESP32, HID]

This video shows an affordable way to add USB support to maker projects. $2 for 10 PCBs (Any Color): https://jlcpcb.com (sponsor) The CH559 is a 8051 architecture based microcontroller that has built in dual USB Hosts. This allows to interface generic Keyboards, Mice, MIDI devices and in my case gamepads. I added this chip as an add on to my ESP32 based game console to replace the SNES controllers by cheap generic ones. The communications is handled over UART which can be done by both ends in hardware. There still is some work to do but you can try it out already. Have fun!
