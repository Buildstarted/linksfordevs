The No-Hassle Code Upgrade Tool for .NET

.NET Core is evolving faster than ever - Visual ReCode brings your projects along for the journey

  • ASP.NET Core was rebuilt from the ground up to be the higest performing, secure web API for modern applications. ReCode can convert your ASP.NET WebAPI solutions to .NET Core, bringing your app logic along preserving your source code investment.

  • Recode generates new gRPC source with the same functionality and data contracts, migrating your application to the latest .NET Core network API with a minimum of effort.

  • Recode generates new Web API source with the same functionality and data contracts, migrating your application to the latest .NET Core network API with a minimum of effort.

Visual ReCode doesn't rely on compatibilty layers or ports of legacy .NET Framework features; it rewrites your application to use the native high performance features in .NET Core.

Once you've ReCoded your application you don't need Visual ReCode around to compile it, modify it, or deploy - because everything's using the native .NET Core implementations just like a clean sheet implementation.
