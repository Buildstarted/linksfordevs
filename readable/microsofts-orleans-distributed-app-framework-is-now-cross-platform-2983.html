Microsoft Orleans is an open source programming framework for .NET, originally built by the eXtreme Computing Group at Microsoft Research that simplifies distributed app development using virtual actors. When Microsoft open sourced Orleans in early 2015, it required the .NET Framework which meant it only ran on Windows Server.
