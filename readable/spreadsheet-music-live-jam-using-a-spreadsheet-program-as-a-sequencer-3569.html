(Spread)sheet Music: Live Jam using a Spreadsheet Program as a Sequencer

by Maxime Chevalier-Boisvert
Watching time: 2 minutes
Posted here: 27 Feb 2019

https://www.youtube.com/watch?v=LQOFiC6YIvo


Just a little hack I made for fun. Using a CSV spreadsheet to sequence music. The code is written in Python and uses the mido package for MIDI output. The audio is produced by a Bass Station II and a Drumbrute. Inspired by: http://cdm.link/2012/01/spreadsheet-as-music-tracker-sequencer-with-libreoffice-nee-openoffice/
