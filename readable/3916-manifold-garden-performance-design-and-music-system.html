Manifold Garden - Performance, Design, and Music System


Dear Fellow Gardener, 

Last December I announced that Manifold Garden was playable from start to finish. Since then we have continued to make improvements to the game across the board. 

A selection of what we've accomplished can be seen in the latest development update video.

Performance
For the most part the game is running at 60FPS on both PC and PS4. We've optimized meshes significantly (in some places reducing the size by 40%) and have smoothed out a lot of the major bumps in the game. Where we are seeing issues is in large levels with highly detailed geometry adjacent to other similarly sized levels. To tackle this we've written several tools to track the memory usage and performance of each level in the game. This has allowed us to evaluate our effectiveness and pinpoint which areas are problematic. It's a slow and tedious process but we're making steady progress. 

Ending Refinement
Getting the game playable from start to finish was a huge milestone for us, but there were still a few areas that were quite rough. Based on feedback from testers the first draft of the ending didn't feel very focused. I've been redesigning the ending using this feedback as well as lessons learned from making the first ending.

Visuals
I had felt for a long time that the visuals of the game were in a really solid state. Nevertheless, we've actually been able to improve on it further - fixing a lot of small details like shadows and making the world wrap transitions smooth.

We also improved the appearance of the edges in the PS4 version significantly.

Music System
We recently redid the music system in Manifold Garden to make it more flexible and modular. The old system was very limited in functionality and the code was quite messy. It was difficult to make changes and add new music. Customization often took several hours.

The new system is much more suitable to the needs of Manifold Garden. It’s organized around the idea of "music sentences", where a piece of music might begin in one scene and then continue throughout others. This creates and enforces a sense of connection between the spaces. It’s also easily customizable and scales very well. New music can be added and implemented in about 30 minutes.

If you’re interested in how the music system works - I did a deep dive video recently where I explain the architecture, the debug tools that we added, and how it’s all implemented in the game.

That's it for this time. I can't announce a ship date yet but we do have a window in mind. Hopefully we will be able to share it with you all soon.
