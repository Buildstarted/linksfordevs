Remove "There were build errors. Would you like to continue and run..." dialog.

Reading time: 2-3 minutes
Posted here: 10 Apr 2020

https://developercommunity.visualstudio.com/idea/985235/remove-there-were-build-errors-would-you-like-to-c.html

Kendra Havens [MSFT] reported 3 hours ago

Over 50 users commented on this tweet that they only ever clicked "yes" here by mistake. https://twitter.com/jerrynixon/status/1248019025235283970

We need to either remove the dialog box or change the default selection to "No."

A few customers gave examples of scenarios where they would click "Yes." I would suggest either the most popular option ("No") should be selected, the dialog is redesigned, or perhaps the dialog should not show unless the user enables an option.
