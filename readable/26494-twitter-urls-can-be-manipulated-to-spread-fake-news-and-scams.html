Twitter URLs Can Be Manipulated to Spread Fake News and Scams

The way Twitter creates URLs to a tweet could be abused for unscrupulous purposes that range from running disinformation campaigns to spreading malware or tricking users into landing on a malicious web page.

What we see in the browser address bar when we access a tweet is a URL that contains a username and the tweet's status ID, which is defined by a long string of numeric characters.

However, the username is completely irrelevant for reaching a specific message on Twitter. Only the tweet-status-id is needed.

For example, the following URLs will all show the same tweet because they contain the same tweet status ID, which in this case is 1087839317534363648.

https://www.twitter.com/davy/status/1087839317534363648
https://www.twitter.com/realDonaldTrump/status/1087839317534363648
https://www.twitter.com/bleepincomputer/status/1087839317534363648

This behavior opens the door to abuse as users could be tricked into thinking that a Twitter user is promoting a particular tweet and its content. This could lead to the spread of disinformation or malicious content as the user clicking on the Tweet thinks it is from a trusted source.

Misdirection through redirection

As Twitter does not care what name is listed in a tweet and only cares about the ID, bad actors can modify tweets to make it appear as if someone is promoting the tweet and content that it redirects to.

Security researcher Davy Wybiral offered a sample of this deception by posting a link purportedly from President Donald Trump, which redirected to an older tweet from the researcher.

This leaves room to well-crafted social engineering that can easily direct to content from an attacker. 

While on desktop browsers the redirect is more visible in the address bar, a deception of this sort can be refined to make it more difficult to detect by a potential victim.

The trick is even more powerful on mobile devices, where the redirect is completely hidden because the tweet opens in the Twitter client.

Using this method, disinformation campaigns can make it look like a political figure, such as President Trump, endorses a particular view by adding his account name to a tweet.

It can even be taken further as Twitter has no restrictions regarding the profile name, an individual with loose moral principles could easily impersonate a Twitter account and use the redirect behavior to point users to shady messages.

To get a better understanding of the risk, BleepingComputer created this shoddy proof-of-concept [twitter.com/NSAGov/status/1139201495856025605] that has the username of the National Security Agency in the link, but the status ID from an account we control.

Opening the link on a desktop computer reveals some clues about the fake account but they may be overlooked if it had a similar name as the impersonated account.

For the purposes of the demonstration, we temporarily changed the profile name and added images to make it look like the account belonged to the NSA.

At the end of the redirect can be anything from false information to links pointing to malware, phishing or other types of scams.

Launching the same link on a mobile phone, though, shows little indication that it is a ploy and there are high chances for the trickery to pass unnoticed.

This redirect manipulation issue is not new. The earliest report we found is from 2012, from Terence Eden, who also offered the info in an answer on Stack Overflow in early January 2015. It was also suggested in 2016 as a solution for retrieving links to specific tweets. And it was reported again in 2017 by freelance developer Bram Van Damme.

Ripe for abuse

Every communication platform is fertile ground for spreading information, even false news. 

Facebook is the typical choice for organizations engaged in shaping distorted opinions of the political, social, or cultural reality. Twitter is no stranger to this, either. Just yesterday, researchers at Recorded Future shared their work on identifying influence operations on social media.

Today, Twitter also announced that it removed almost 5,000 fake accounts registered for state-backed activities. This is just a drop in a bucket that has been filling for quite some time with fake accounts that generated over 30 million tweets and published more than one terabyte of media files.

Wybiral told BleepingComputer that one method to abuse Twitter is to start using handles of accounts that are no longer active. This works because Twitter allows changing the account username, anyone can pick one that was abandoned. A username that was also referenced in the past by a reputable source that is still available has a greater chance to pass the legitimacy test.

"So a clever troll can scrape the web for account links that aren't active anymore and start using their handle. If they need proof that they're a real person and not a troll... They can cite the older reference to the account," the researcher told us.
