Support Announcements

We now support Windows Server 2019 and Windows Server 1809 for Azure and Standalone Service Fabric Clusters starting from 6.4.654.9590. 

Breaking Changes in Upcoming 6.5 Release

In Service Fabric version 6.5, we will make some breaking changes to the node state removal interfaces. This includes FabricClient.ClusterManagementClient.RemoveNodeStateAsync (.NET API), Remove-ServiceFabricNodeState (PowerShell), Remove Node State (REST), sfctl node remove-state (CLI). If you are using any of these interfaces, please read the release notes for Service Fabric version 6.4 CU5 for more information on this breaking change and how you can prepare for it.

For more details, please read the release notes.
