VisualSOS.Extension - Visual Studio Marketplace

Reading time: 1 minute
Posted here: 13 Aug 2020

https://marketplace.visualstudio.com/items?itemName=AngelHernandezMatos.VisualSO-Extension-1-0

Visual SOS is both an executable and extension for Visual Studio that leverages the debugger engine available in Windows in conjunction with SOS debugging extension.

It’s similar to having Windbg embedded into Visual Studio but in a much simpler way, because it exposes some of the most commonly used commands just a click away.

It’s free and open-source (GPL), developers are encouraged to collaborate on it.
