Welcome

Welcome to The .NET Core Podcast Newsletter. The only newsletter to which will provide you with fortnightly updates on the only podcast which is devoted to:

  • .NET Core
  • ASP.NET Core
  • EF Core
  • SignalR

and not forgetting The .NET Core community, itself.

Content You Can Expect

Each edition of the newsletter will have:

  • The bio of the coming episode's guest, or a description of the monologue episode due out
  • Links to some of the places where you can subscribe to the show
  • A number of interesting community links to check out
  • Ways to connect with the show
  • A sponsor message (gotta keep the lights on)

So lets sit back, open up a terminal, type in dotnet new newsletter and let the show begin.
