We'll have a fairly relaxed agenda, but will follow:
- 👋 Introduction / Housekeeping
- 🗣️ Round table introductions
- 👨‍🏫 Learn to Code with Layla Porter / Kevin Smith
- 🎉 Closing
- 🍻 Social

👉 **Learn to Code**

This is intended for absolute beginners with a willingness to learn a new skill.

This session will cover the fundamentals of Git and GitHub, but people can still hack on C# and get help from mentors.

All you need is a laptop. (Windows, Linux or iOS - any operating system is fine)

We will have food and drink to keep you going!
