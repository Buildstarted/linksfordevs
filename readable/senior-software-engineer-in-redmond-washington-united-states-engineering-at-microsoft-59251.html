Responsibilities

Design/Architect/Build/Maintain features, services and entire systems. Primarily for use as part of the infrastructure of Developer Relations web experiences such as docs.microsoft.com and azure.microsoft.com. Work will primarily be .NET and deployed to Azure.

Qualifications

Requirements

  • 5+ years experience with C# and .Net platform technologies as well as the associated web frameworks (ASP.Net MVC, Web API)
  • Experience using git or other distributed version control system
  • A passion for building solid systems that includes Test Driven Development and strong knowledge of architectural principles and models such as SOLID, Event Sourcing, REST, microservices, etc.
  • Ability to build and evolve systems that can be stood up quickly but also able to stand the test of time in the face of uncertainty
  • Experience working in an agile, live site/devops culture that deploys code frequently and automates everything from tests to builds to infrastructure deployment

Preferred Skills

  • Experience with .Net Core technologies as well as ASP.Net Core
  • Experience with Node.js, JavaScript, and front-end web development
  • Experience working with the Microsoft Azure platform or industry equivalent
  • Experience with xunit for unit testing
  • Experience with Azure DevOps work item tracking, build, and release systems
  • Experience with Azure Resource Manager templates or other “infrastructure as code” systems
