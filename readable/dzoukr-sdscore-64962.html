Dzoukr/SDSCore

by Dzoukr
Reading time: 1 minute
Posted here: 29 Jul 2020

https://github.com/Dzoukr/SDSCore

This branch is 7 commits ahead of predictionmachines:master.

This is a .NET Standard 2.0 / .NET Core 3.1 port of amazing SDSLite library for manipulating netCDF, CSV and TSV files.

All credit goes to contributors of the original library, I just did few cleanups and small refactoring regarding reading library paths.

Please see the original library documentation for understanding how to read netCDF files and how to setup necessary dependencies.
