NTRU Prime: NIST submission

NTRU Prime is a second-round candidate in NIST's Post-Quantum Cryptography Standardization Project. The second-round submission package specifies six KEMs:

  • Streamlined NTRU Prime: sntrup653, sntrup761, sntrup857.
  • NTRU LPRime: ntrulpr653, ntrulpr761, ntrulpr857.

First-round submission

The first-round submission package specified two KEMs:

  • sntrup4591761. Uses same one-way function as sntrup761.
  • ntrulpr4591761. Uses same one-way function as ntrulpr761.
