Guidance for Java EE Migration to Azure

Reading time: 1-2 minutes
Posted here: 30 Mar 2020

https://microsoft.qualtrics.com/jfe/form/SV_b903dEDAOzbiXDT

The Azure team at Microsoft is actively working to improve our guidance for moving workloads running on Java EE application servers to Linux virtual machines, Azure Kubernetes Service (AKS), and Azure Red Hat OpenShift.

In this 5 minute survey, we would like to learn about your Java EE migration plans and guidance needs.

If you are willing to have a follow-up conversation with us to help us better understand your scenarios to ensure we are fully addressing them, please include your contact information at the end of the survey. The program managers, architects and engineers responsible for developing the guidance would like to work as closely as possible with you to ensure a smooth migration experience from beginning to end.

********* 

All data from this survey will be anonymized. (You can explicitly opt in to provide your contact information voluntarily at the end of the survey if you are interested in follow-up with our engineering team). See the Microsoft Privacy Policy for more information.
