The .NET Core Windows Forms visual designer is now available in Visual Studio 16.5 Preview1 or later. You can enable the WinForms .NET Core designer in Tools -> Options -> Environment -> Preview Features with Use the preview Windows Forms designer for .NET Core apps.

Releases of .NET Core Windows Forms Designer are tied to Visual Studio Preview releases, so update to the latest Visual Studio Preview version to get the latest designer features.
