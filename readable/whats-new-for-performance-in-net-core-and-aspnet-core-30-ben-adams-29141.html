What’s new for performance in .NET Core and ASP.NET Core 3.0 - Ben Adams

Speaker: Ben Adams (Illyriad Games / Microsoft MVP / .NET Foundation Board Director) Session outline: One of the biggest advantages of using .NET Core (besides cross-platform support) is the drastic improvements in performance. Because the .NET Core team was able to make minor breaking changes in the runtime and Base Class Library (BCL), lots of stuff was implemented much more efficiently. In this session Ben will dive into the performance improvements in .NET Core in the 3.0 release: runtime changes, JIT changes, intrinsics and a deep dive into some of the improvements making it the best release yet!
