Resolution The HTML width is set to 980 pixels by default. By entering <meta name="viewport" content="width=320" /> within the HTML <head>...</head>, the width will be set to 320 pixels and there will be no zoom feature.
Image Display/Saving

Images will display independently by either touching the image and holding for a few seconds or touching the URL for the JPEG image. Images can be saved to a microSD card. Saved images can be viewed in Nintendo 3DS Camera.

Additionally, MPO format 3D images shot with Nintendo 3DS Camera can be displayed independently (3D) and saved.

Some images cannot be displayed/saved depending on the website and/or image size.
Uploading Images (Sending)

Images saved on the device or microSD card such as images shot with Nintendo 3DS Camera can be uploaded on compatible websites.

  • Saving and uploading images and photos can be restricted through Parental Controls on the console.
  • Videos and audio files cannot be uploaded.
