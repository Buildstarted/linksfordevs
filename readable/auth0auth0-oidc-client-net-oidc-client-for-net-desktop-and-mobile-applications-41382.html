auth0/auth0-oidc-client-net

by damieng
Reading time: 1 minute
Posted here: 10 Nov 2019

https://github.com/auth0/auth0-oidc-client-net

OIDC Client for .NET Desktop (WinForms, WPF and UWP) as well as Xamarin-based iOS and Android mobile applications.

This library makes use of the the IdentityModel/IdentityModel.OidcClient library and uses code from the IdentityModel/IdentityModel.OidcClient.Samples repository to achieve browser integration.

For information on how to use this library, please refer to the relevant Quickstarts:

  • UWP
  • WPF/WinForms
  • Xamarin

and also to our docs for this library.

Breaking changes in v3

Version 3 of this library includes many potential breaking changes depending on your usage. Please read Migrating to Auth0-OIDC-Client-NET v3.
