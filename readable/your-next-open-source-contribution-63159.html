Today Open Source is ubiquitous and comprises nearly 80% – 90% of the code in a “typical” application. There is a need to use Open Source to stay competitive, but what about the need to use open source to advance your career? In this talk, I will talk through actionable ways to make open-source contributions to grow your career. --- Brian Douglas is a Developer Advocate at GitHub where he works on increasing use of the GitHub API by iterating on the onboarding process and helping users with evergreen technical content.
