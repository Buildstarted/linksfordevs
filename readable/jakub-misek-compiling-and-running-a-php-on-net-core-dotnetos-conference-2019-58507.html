Visit our website to learn more! https://conf.dotnetos.org/ Facebook: https://www.facebook.com/dotnetos/ Twitter: https://twitter.com/dotnetosorg Jakub Míšek is a veteran .NET developer and co-founder of iolevel, where he is maintaining and developing the PeachPie compiler platform. Jakub holds a Master's degree in Software Engineering from the Charles University in Prague, where he is also pursuing his PhD. C# and the .NET Framework have been providing the support for dynamic objects for years already. The reflection API has been improved significantly in .NET Core. So what if we put all this together in order to improve a real dynamic language by running it on top of .NET? The compiler and the runtime have to work together. Also, we have to find a way of implementing the rest of the language’s dynamic features in the most performant way. Let’s talk about how “dynamic” actually started thanks to PHP, how to implement the dynamicity with a custom made compiler on Roslyn, and what are the achievements of such an approach.
