This is the recording of the 2020 edition of .NET Core Summer Event. The event is a one day mini-conference with an international speaker line-up (including 8 MVPs) focused on software development with .NET Core. Check out the event website: https://dncse.nl Thanks to our sponsors for making this event possible: - Info Support (https://www.infosupport.com) - DotNed, the Dutch .NET User Group (https://www.dotned.nl). See the list of sessions below. You can click each timestamp to skip to that specific session. The first part of the stream (about 1 hour) contains only a splash-screen that showed while attendees were joining the live stream.
