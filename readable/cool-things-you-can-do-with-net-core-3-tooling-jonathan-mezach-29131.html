Cool things you can do with .NET Core 3 Tooling - Jonathan Mezach

by NET Core Summer Event
Watching time: 47 minutes
Posted here: 10 Jul 2019

https://www.youtube.com/watch?v=MSCRzPiGKr8


Speaker: Jonathan Mezach (Info Support) Session outline: With .NET Core a lot has changed in terms of tooling. From making it easier to get started, to working together in teams and documenting shared libraries all the way to diagnosing (performance) issues in your apps in production. In this demo heavy session I’ll show a sample of all these new tools that will be released with .NET Core 3 and how to use them. After this session you’ll have learned something new about the tools you’re already using, or perhaps an entirely new tool that you can add to your tool-belt.
