The Raspberry Pi Power over Ethernet HAT is a small accessory for the Raspberry Pi computer. It can only be used with the Raspberry Pi 3 Model B+ or the Raspberry Pi 4. The PoE HAT allows you to power your Raspberry Pi using Power over Ethernetâ€“enabled networks; for this product to be used, the network it is connected to needs to have power-sourcing equipment installed.

Compatible with Raspberry Pi 3 B+ only.

Requires:

  • A Raspberry Pi 3 Model B+ or Raspberry Pi 4 Model B
  • An Ethernet cable
  • Power-sourcing equipment for a 802.3af Power over Ethernet network
