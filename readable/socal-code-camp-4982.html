Jon Bachelor

8:45 AM - Saturday, November 10, 2018

Xamarin | UWP | prism | MVVM | mobile development | iOS | Android

Location: SGM 101

Xamarin.Forms is an exciting mobile development platform where an Android, iOS, and even UWP app can all share the vast majority of both back-end and front-end code. We'll learn how to build an app using the Prism framework to help handle dependency injection, navigation, and more!

The Prism framework combined with a dependency injection container is a powerful ally when building an app using an MVVM (Model, View, ViewModel) architecture. The framework helps us "fall into the pit of success" by encouraging loose coupling and highly-testable code. We'll implement a basic app designed to demonstrate how you too can start creating native apps to release on multiple platforms with nearly all code shared. All code from the demo will be shared on GitHub as sample code to work from. Come have some cross-platform fun with Xamarin!
