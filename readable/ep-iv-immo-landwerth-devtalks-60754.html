Anything .NET Core, .NET Standard, .NET 5

Unser erster Gast bei DevTalks ist Immo Landwerth.

Immo arbeitet als Program Manager im .NET Plattform Team. Sein Fokus liegt dabei auf der Base Class Library (BCL), .NET Standard und API Design. Außerdem moderiert er regelmäßig bei YouTube Design Reviews für die .NET Foundation.

Wir werden mit Immo über seine Arbeit bei Microsoft, das kommende .NET5 und was nun eigentlich mit .NET Standard passiert ist sprechen.
